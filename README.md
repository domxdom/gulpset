# gulpset

Gulp based project skeleton with modular tasks.

- `gulpset init`

| key | value |
| :-- | :-- |
| js | `esNext`(default), `es5`, `ts` |
| css | `scss`(default), `stylus` |
| html | `ejs`(default), `html`, `pug` |

linterは、利用する環境に応じたものが自動追加

- TypeScriptでGulpfileを記載する
